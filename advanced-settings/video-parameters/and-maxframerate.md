# \&maxframerate

## Aliases

* `&mfr`
* `&mfps`

## Options

| Value                    | Description                    |
| ------------------------ | ------------------------------ |
| (positive integer value) | FPS value in frames per second |

## Details

Like [`&framerate`](../../advanced-settings.md#framerateframe-rate), except it will allow for lower frame rates if the specific frame rate requested failed.
